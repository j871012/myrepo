開啟方式	開啟命令提示字元
		移至此專案資料夾內
		終端機輸入 npm install
		終端機輸入 npm start

操作步驟:

http://127.0.0.1:7000 即可進入首頁
進入首頁，可選擇註冊帳號密碼或是登入使用者
登入帳號密碼之後，可以選擇買家或是賣家
選擇賣家後可以上架商品及創建房間
上架商品可以選擇指定的金額、數量及圖片
創建房間時會有指定的串流金鑰，金鑰使用在OBS上
OBS設定串流分類，伺服器輸入rtmp://vm214.ims.ice.cyut.edu.tw:9080/live
串流金鑰則是創建房間時由網頁上提供
設定完畢後即可開始串流直播！
選擇買家後可以進入到房間列表介面
可選擇自己喜愛的房間點擊進入
即可觀看直播且可與直播主和觀眾聊天互動
觀看直播時且想要購買商品，即可點選"購物去"按鈕到商品列表
之後選擇自己需要的商品及數量，加入購物車後即可結單
